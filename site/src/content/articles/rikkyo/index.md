# 立教大学の志望理由｜学部別・入試方式の完全ガイド

「立教の志望理由書、何を書けばいい？」——立教大学はキリスト教（聖公会）を建学の精神に掲げ、**「自由の学府」として知・徳・美の全人教育**を実践する大学です。11学部＋GLAPという多彩な構成で、学部によって求める学生像が根本から異なります。

特に立教固有の入試として**自由選抜入試**（学校長推薦不要・評定要件なし・活動実績重視）があり、指定校推薦と評価軸が大きく違います。「立教の雰囲気が好き」「池袋キャンパスに憧れる」では両方式とも通過できません。

## この記事のポイント

- 立教の志望理由は「**建学精神（奉仕・共生）× 学部AP × 自分の問い**」の3軸
- **自由選抜入試**は指定校推薦より「主体的活動実績×研究テーマの具体性」が重視される
- 観光学部・異文化コミュニケーション学部・GLAP・コミュニティ福祉学部など**他大学にない固有学部**が差別化軸
- 「池袋キャンパスが好き」「MARCHだから」は全方式で落ちる

## 立教大学の建学精神と入試の特徴

### キリスト教（聖公会）建学精神

立教大学は1874年、米国聖公会の宣教師チャニング・ムーア・ウィリアムズによって創立されました。建学の精神は**「神の前に誠実に生き、他者と共に社会に仕える人材の育成」**です。

この精神は推薦入試の評価軸にも反映されています。「自分だけが成功する」ではなく「**他者・社会への貢献意識と、それを実現するための専門知識**」を持つ受験生が評価されます。

### 自由選抜入試 vs 指定校推薦

| 項目 | 自由選抜入試 | 指定校推薦 |
|------|-----------|---------|
| 学校長推薦 | 不要 | 必要 |
| 評定要件 | なし〜低い | 3.8〜4.0程度 |
| 主な評価軸 | 活動実績＋研究テーマの主体性 | 学校成績＋志望理由書＋面接 |
| 志望理由書 | 研究テーマ×活動実績の接続が必須 | 志望動機の明確さ＋学習計画 |

## 学部タイプ診断

**A. 文学・哲学・キリスト教思想・教育を深く探究したい** → **文学部**（キリスト教学科／文学科／教育学科）

**B. 経済学・経営学・会計ファイナンスを研究したい** → **経済学部**（3学科）

**C. 経営を英語環境で学びたい・国際ビジネスに取り組みたい** → **経営学部**（国際経営学科含む）

**D. 数学・物理・化学・生命科学の基礎研究をしたい** → **理学部**（4学科）

**E. 社会問題・メディア・文化を実証的に研究したい** → **社会学部**（3学科）

**F. 法律・国際ビジネス法を研究したい** → **法学部**（法学科／国際ビジネス法学科）

**G. 観光・ホスピタリティ・異文化交流を専門的に研究したい** → **観光学部**（観光学科／交流文化学科）

**H. 福祉・コミュニティ政策・スポーツウエルネスを研究したい** → **コミュニティ福祉学部**（3学科）

**I. 心理学・映像身体論を研究したい** → **現代心理学部**（心理学科／映像身体学科）

**J. 英語×異文化コミュニケーションを専門的に学びたい** → **異文化コミュニケーション学部**

**K. 英語完全授業でリベラルアーツを学びたい（少人数精鋭）** → **GLAP**

## 受かる理由⇔落ちる理由

### 軸1：「立教でなければならない理由」——全学部共通

| 受かる志望理由 | 落ちる志望理由 |
|-------------|-------------|
| 「観光学部の交流文化学科は、観光×文化人類学の学際研究ができる国内でも稀有な環境」 | 「池袋キャンパスで勉強したかった」 |
| 「GLAP の少人数・全英語環境でしか実現できない問いがある」 | 「英語が得意だからGLAPを受ける」 |

### 軸2：「建学精神との接続」——立教固有の評価軸

| 受かる志望理由 | 落ちる志望理由 |
|-------------|-------------|
| 「コミュニティ福祉学部で学ぶ支援技術を、将来は地域の孤立した高齢者へのアウトリーチに活かす——これが立教の『奉仕』精神の実践」 | 「立教のキリスト教の雰囲気に憧れました」 |

### 軸3：「自由選抜 vs 指定校」——方式による書き分け

| 方式 | 志望理由書の重点 |
|------|--------------|
| 自由選抜 | 「高校時代の主体的な活動実績→問いの形成→入学後の研究計画」の一貫した物語 |
| 指定校推薦 | 「なぜ立教のこの学部か」の明確な志望動機＋学習計画の具体性 |

## よくある質問（FAQ）

**Q1. MARCHの中で立教を選ぶ積極的な理由は何ですか？**

立教固有の強みとして：①**観光学部**（観光学専門学部として国内有数の歴史）、②**異文化コミュニケーション学部**（英語×異文化の専門学部）、③**GLAP**（全授業英語の少人数精鋭プログラム）、④**コミュニティ福祉学部のスポーツウエルネス学科**、⑤**現代心理学部の映像身体学科**（映像×身体論）——が他大学にない差別化軸です。

**Q2. 立教の自由選抜入試は何が指定校推薦と違いますか？**

最大の違いは「学校長推薦が不要」「評定要件がほぼなし」という点です。代わりに「高校時代に何を主体的にやってきたか（活動実績）」と「それが立教のこの学部での研究テーマとどう繋がるか」の具体性が強く問われます。

**Q3. 立教と明治・青山の志望理由書はどう書き分けますか？**

- **立教**：キリスト教建学精神（奉仕・共生）との接続＋固有学部（観光・異文化・GLAP）の活用
- **明治**：権利自由の精神＋地域行政・公共経営など明治固有学科
- **青山**：キリスト教系ではあるが青山らしさは「国際性×都市」の軸

**Q4. GLAPと異文化コミュニケーション学部はどう違いますか？**

| | GLAP | 異文化コミュニケーション学部 |
|---|-----|----------------------|
| 授業言語 | 全授業英語 | 英語科目中心（日本語も一部） |
| 定員 | 約50名/年（少人数精鋭） | 通常規模 |
| 学びの内容 | リベラルアーツ全般（学際） | 異文化コミュニケーション専門 |
| 志望理由の核心 | 英語×学際的問いの必然性 | 英語×異文化研究の問い |
