# 法政大学の志望理由｜学部別・推薦入試の完全ガイド

「法政大学の志望理由書、何を書けばいい？」——法政大学は「**自由と進歩**」を建学精神に掲げ、学生の自律的な学びと社会変革への意志を育てる大学です。

法政大学の最大の特徴は**他MARCHにない固有学部**が複数存在すること。**キャリアデザイン学部**（キャリアを学術的に研究）・**グローバル教養学部（GIS）**（全授業英語）・**スポーツ健康学部**（スポーツ×健康×福祉）は他大学に同名学部がほとんどなく、法政の最大の差別化軸です。

## この記事のポイント

- 法政の志望理由は「**「自由と進歩」建学精神 × 学部AP × 固有学部の問い**」の3軸
- **キャリアデザイン学部・GIS・スポーツ健康学部**は他MARCH・上智にも少ない固有学部
- 「法政の雰囲気が好き」「都心キャンパスだから」は全方式で落ちる
- 固有学部を活かせる「問い」を持つ受験生には法政が最も差別化しやすい

## 法政大学の建学精神と入試の特徴

### 「自由と進歩」の建学精神

法政大学は1880年（明治13年）、東京法学社として創立されました。建学精神「**自由と進歩**」は、個人の自由な思想・学問の追究と、社会の進歩への貢献という二つの価値観を体現しています。

推薦入試の評価軸にも「**自分の問いを自由に設定し、社会の進歩に繋げる研究への意志**」が反映されています。「与えられた問いに答える」ではなく「自分で問いを立てる」姿勢を持つ受験生が評価されます。

### キャンパス構成

| キャンパス | 主な学部 |
|----------|---------|
| **市ヶ谷キャンパス**（都心） | 法・文・経営・国際文化・GIS・現代福祉・キャリアデザイン・デザイン工学等 |
| **多摩キャンパス** | 社会・経済・スポーツ健康・現代福祉（一部） |
| **小金井キャンパス** | 情報科学・理工学・生命科学 |

### 推薦入試の種類

| 方式 | 特徴 |
|------|------|
| **指定校推薦** | 学校長推薦必要・評定基準あり |
| **公募推薦** | 出願資格を満たせば出願可 |
| **自己推薦（スポーツ推薦等）** | 活動実績重視・学部による |

## どのタイプが法政向きか：3タイプ診断

**タイプ1：「キャリア」を学術的テーマとして研究したい**
→ **キャリアデザイン学部**（キャリア形成・働き方・職業社会を学術的に研究——他大学に同名学部がほとんどない）

**タイプ2：英語で社会科学・人文学を学びたい**
→ **グローバル教養学部（GIS）**（全授業英語・少人数・社会科学×人文系リベラルアーツ）

**タイプ3：スポーツを健康・社会政策・福祉の視点で研究したい**
→ **スポーツ健康学部**（スポーツ科学×健康×社会政策——体育系専門学校とは異なる学術的アプローチ）

その他：法学部（伝統的な強み）、経営学部、文学部（心理系含む）

## 受かる理由⇔落ちる理由

| 受かる志望理由の特徴 | 落ちる志望理由の特徴 |
|-----------------|-----------------|
| 「キャリアデザイン学部でなければ研究できない問い」が明示されている | 「キャリアに関心がある」「就職を考えて選んだ」 |
| GIS志望者：「英語で社会科学を学ぶ必然性がある問い」がある | 「英語が得意だから」「グローバルに活躍したい」だけ |
| スポーツ健康学部：「スポーツ×社会・健康の研究問い」がある | 「スポーツが好き」「アスリートとして貢献したい」だけ |
| 建学精神「自由と進歩」と問いの接続がある | 「法政の雰囲気が好き」「都心キャンパスだから」 |

## よくある質問（FAQ）

**Q1: 法政大学と立教大学、どちらもMARCHだが志望理由書はどう変わるか**

立教はキリスト教（聖公会）建学精神と「自由選抜入試（活動実績重視）」が特徴。法政は「自由と進歩」建学精神と固有学部（キャリアデザイン・GIS・スポーツ健康）が差別化軸。「法政固有の学部でなければ研究できない問い」を軸にすると、「なぜ立教でなく法政か」への回答になります。

**Q2: キャリアデザイン学部は「就職に有利になるための学部」か**

キャリアデザイン学部は「就職の仕方を学ぶ」学部ではなく、「キャリア・働き方・職業社会を学術的に研究する」学部です。「就職に有利になりたいから」という動機では落ちます。「なぜ人はキャリアをデザインできるのか・できないのか」という学術的問いが必要です。

**Q3: GIS（グローバル教養学部）は定員が少ないが難易度が高いか**

GISは定員が少なく少人数教育が特徴です。英語での学術的能力（英語論文読解・英語ディスカッション・英語ライティング）が前提とされます。「英語で社会科学・人文学を研究する必然性がある問い」と英語力の証明（スコア・英語活動実績）が重要です。
