# 総合型選抜情報サイト

## 役割
- 事業責任者兼PdM兼マーケティング担当

## 概要
- AIによる書類添削サービスのサイト。総合型選抜や推薦入試（旧AO入試）に関する情報も発信。
- サービス名: オープン添削
- 早慶上智MARCHを中心とした私立上位校を目指す高校3年生が対象

## 検索キーワード
- 「大学名+志望理由」（例：早稲田　志望理由）
- 「大学名+学部学科名+志望理由」（例：上智 国際グローバル 志望理由）

## 構成
- `site/` : サイト本体（React JSX + HTML）
- `site/src/content/articles/` : 記事コンテンツ（Markdown・マスター）
- `assets/` : ロゴ・画像素材
- `docs/` : 事業計画書・記事リスト等のドキュメント
- `docs/templates/` : 記事制作の標準テンプレ・SEOチェックリスト

## ヘッダー運用
- SPAトップ、記事一覧、総合型選抜診断のヘッダーは共通状態を維持する
- 共通ヘッダーのリンクは「受験対策記事」(`/articles`)、「総合型選抜診断」(`/match/`)、「無料で添削する」(`/#step1Sec`)
- ヘッダー変更時はPC表示とモバイルメニューの両方を同じ構成に揃える

## 監修者プロフィール（記事冒頭・末尾に必ず掲載）
- 新卒採用10年・1,000人以上の志望動機を評価｜総合型選抜専門塾4年・100名以上指導｜早慶上智MARCH多数合格
- 「採用担当者の視点」と「総合型選抜の指導知見」の二刀流が差別化軸
- 監修者名は当面非公開
- 詳細プロフィール：監修者プロフィールはパターンA（信頼感・権威性重視）を採用

## 記事制作フロー
1. 記事タイプに応じてテンプレを使用
   - ハブ記事（大学名+志望理由）：`docs/templates/article-hub-template.md`
   - 学部別記事：`docs/templates/article-faculty-template.md`
   - 英語入試記事：`docs/templates/article-eigo-template.md`
2. 公開前は `docs/templates/seo-checklist.md` でチェック
3. ターゲットKW優先順位：早稲田S群（政経/SILS/商/法）→A群→B群→C群（理工系・推薦特化）→慶應→上智→MARCHの順で横展開

## SEO戦略の前提
- 新規ドメインのため、ロングテール（学部別）から積みトピカルオーソリティ構築
- ビッグKW（「早稲田 志望理由」等）はハブ記事＋学部別記事のクラスター構造で6〜12ヶ月かけて獲得
- E-E-A-T重視：監修者プロフィール・公式情報の出典明記・一次データ（合格者アンケート等）を必須化
