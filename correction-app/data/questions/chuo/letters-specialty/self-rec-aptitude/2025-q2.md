# 中央大学 文学部 自己推薦特別入試（専攻適性型） 2025年度 小論文・自己推薦文

## 入試概要

- 入試種別: 総合型選抜（自己推薦特別入試 専攻適性型）
- 1次選考: 書類審査（志望理由書・志願者経歴書・小論文・活動内容説明資料等）
- 2次選考: 専攻別試験（90分）+ 面接
- 評定要件: 不問
- 英語資格要件: 不問
- 専願/併願: 専願

## 設問

**小論文（書式3）／自己推薦文【専攻別・1次選考に提出】**

専攻ごとに指定テーマで記述（手書きまたはPC作成可）。

| 専攻 | テーマ（字数） |
|------|--------------|
| フランス語文学文化専攻 | 美術館における活動を通じて得たものについて論じなさい。（4,000字程度） |
| 日本史学専攻 | 指定図書（5冊から1冊選択）の内容を要約した上で、注目点・疑問点・論点を提示しながら著者の見解と自分の意見を交えて論述しなさい。（4,000字程度）※野口実、沖本幸子、森幸夫、呉座勇一、村井章介の著書から選択 |
| 哲学専攻 | 今までに読んだ哲学書について論じなさい。（3,000字程度） |
| 社会学専攻 | 自己推薦文（1,500字程度）：自分がなぜ本専攻に入学するにふさわしいかを論じなさい。※客観的な自己分析（活動・学習実績・体験の具体的提示）と、本専攻で学びたい社会への関心事（社会問題・社会現象等）への言及を必ず含めること |
| 心理学専攻 | 科学的な根拠を提示している心理学の本（2冊以上）を取り上げ、その長所と問題点について論じなさい。（3,000字程度） |
| 学びのパスポートプログラム | 大学を卒業するときに、学びのパスポートプログラムを通じてどのような力を身につけたいのか、これまでの経験や学習を振り返りつつ、今後どのような大学生活を送る計画があるか具体的に述べなさい。（2,000字程度） |

**2次選考：専攻別試験（90分）**

| 専攻 | 試験内容 |
|------|---------|
| フランス語文学文化専攻 | 講義理解力試験（視覚資料を用いた演習形式のディスカッション） |
| 日本史学専攻 | 歴史資料読解力試験（歴史資料に関する講義後、資料読解力試験） |
| 哲学専攻 | 資料読解力試験（哲学者の英語文章を読み、内容について日本語で論述） |
| 社会学専攻 | 講義理解力試験（教員の講義受講後、内容に関する小論文試験） |
| 心理学専攻 | 資料読解力試験（資料を読んだ上で内容に関する小論文試験） |
| 学びのパスポートプログラム | 読解力試験（学問領域を横断するテキスト・資料等を読んで論述） |

## 攻略のポイント

- 1次選考の小論文が実質的に合否を左右する重要書類。専攻別テーマに合わせた入念な準備が必要
- 日本史学専攻は指定図書を事前に読んでおくことが必須（全5冊の中から1冊選択）
- 哲学専攻・心理学専攻は自分で選んだ書籍について論じる形式のため、読書歴の幅と論述力が問われる
- 社会学専攻の「自己推薦文」は客観的な自己分析と、社会問題への学術的な関心の明示が評価軸
- 2次選考の専攻別試験は当日初見の資料・講義への対応力を問う形式
