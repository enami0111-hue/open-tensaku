# 慶應義塾大学 文学部 自主応募制推薦 2025年度 自己推薦書

## 入試概要

- 入試種別: 学校推薦型選抜（公募・自主応募）
- 1次選考: 書類審査（自己推薦書・調査書等）
- 2次選考: 総合考査Ⅰ（120分）+ 総合考査Ⅱ（60分）
- 評定要件: 全期間の学習成績の状況が 4.1以上
- 英語資格要件: 不問
- 専願/併願: 専願（合格した場合は入学を確約）

## 設問

**自己推薦書（本学所定用紙・A4：2ページ）**

以下の2問に対して、各450〜500字程度を目安に記述する：

**問1：**
「高等学校で何に力を入れ、どのような成果を上げましたか。具体的に書いてください（資格を取得した場合は、そのことについても触れてください）。また、可能な限り、それらを証明する書類（コピー可）を添付してください。」

**問2：**
「文学部をなぜ第一志望としたのですか。また、文学部で何を学び、将来、どのように活かそうと考えていますか。具体的に書いてください。」

## 攻略のポイント

- 文学部の入学者受け入れ方針（AP）に基づき、「幅広い人文学への強い関心」「主体的な学習姿勢」が重視される
- 問1は活動実績の具体性・証明可能性が重要
- 問2は「なぜ他大学ではなく慶應文学部か」「文学部のどの専攻・分野か」を明確に
- 総合考査Ⅰは長文読解（日本語）、総合考査Ⅱは論述（テーマ型）。自己推薦書と試験内容が連動することがある
